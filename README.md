# Intelligent Banking Assistant (AI Chatbot)

## Project Description

This project is a rule-based Artificial Intelligence banking assistant chatbot.
The system simulates a bank customer support agent that can answer queries, make decisions and learn new knowledge.

## Features

* Customer login using PIN
* Answers banking queries
* Loan eligibility checking
* Credit card approval decision
* Deposit and withdrawal logging
* Learns new questions automatically
* Knowledge stored permanently

## AI Concepts Used

* Intelligent Agent
* Knowledge Representation
* Inference Engine
* Learning Agent
* Goal Based Agent
* Utility Based Agent

## Technologies Used

* Python
* Text based database

## How to Run

1. Install Python
2. Download project
3. Open terminal in project folder
4. Run:

```
python bankbot.py
```

## Example Questions

* saving account
* upi limit
* loan types
* deposit cash
* branch timing
